# CleanCalculator

**CleanCalculator** is a lightweight, responsive web-based calculator built with pure HTML, CSS, and JavaScript. It provides a clean user interface and supports both mouse clicks and keyboard shortcuts for efficient calculations.

---

## Tech Stack

- **HTML** – Structure of the calculator UI.
- **CSS** – Styling and responsive layout.
- **JavaScript** – Core functionality, event handling, and calculations.

---

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, division.
- Real‑time display of the current expression and result.
- Keyboard support:
  - Numbers `0-9` and decimal point `.`
  - `Enter` to evaluate the expression
  - `Esc` to clear (All Clear)
  - `Backspace` to delete the last character
- Responsive design that works on desktop and mobile browsers.
- Clear visual feedback on button presses.
- Simple, maintainable code structure split across `index.html`, `styles.css`, and `script.js`.

## Usage

### Button Layout
- The calculator displays a grid of buttons for digits `0‑9`, a decimal point, arithmetic operators (`+`, `-`, `*`, `/`), an equals sign (`=`), a clear button (`AC`), and a delete button (`⌫`).
- The display area at the top shows the current expression and the computed result.

### Keyboard Shortcuts
- **Numbers & Decimal**: Press `0‑9` or `.` to input digits.
- **Operators**: Use `+`, `-`, `*`, `/` for arithmetic operations.
- **Enter**: Evaluate the current expression (same as clicking `=`).
- **Esc**: Clear the entire expression (All Clear).
- **Backspace**: Delete the last character (same as clicking `⌫`).

### Responsive Behavior
- On smaller screens (e.g., mobile devices), the button grid scales down while maintaining touch‑friendly sizes.
- The layout automatically adjusts to portrait and landscape orientations.
